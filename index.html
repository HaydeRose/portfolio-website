Trico

The design process of Trico followed the User-Centered Desing. Starting by conducting a competitive analysis and creating personas. Moving on to ideating user flows, a sitemap, and wireframes. And lastly, the high-fidelity prototype was built using Adobe XD and user testing was conducted to get real user feedback.
